A modern weather dashboard built with **Python** and **Streamlit** using the "Vibe Coding" methodology (AI-Assisted Development).

![App Screenshot](screenshot.png)

## Key Features (3-5 Features)
1.  **Real-Time Weather:** Fetches live temperature, wind, and humidity from Open-Meteo API.
2.  **Interactive Forecast:** Displays a 7-day temperature trend using Plotly charts.
3.  **City Search:** Supports geocoding to find cities globally.
4.  **Search History:** Remembers the last searched cities in the sidebar (Session Persistence).
5.  **Error Handling:** Gracefully handles invalid city names.

## Setup Instructions

To run this project locally:

1.  **Clone the repo:**
    ```bash
    git clone <https://github.com/BerkayTurkozx1/weather-dashboard-vibe>
    cd weather-app
    ```

2.  **Install dependencies:**
    ```bash
    pip install -r requirements.txt
    ```

3.  **Run the app:**
    ```bash
    streamlit run app.py
    ```
